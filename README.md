# Data Science & Machine Learning Case Studies

**UT Austin McCombs School of Business | Post Graduate Program in Machine Learning & AI**

This collection showcases foundational data science and machine learning work completed as part of my Post Graduate Program at UT Austin's McCombs School of Business. These projects were completed **without AI assistance** as they were pre-ChatGPT/OpenAI.

---

## 📚 About This Collection

These case studies represent core competencies in:
- Exploratory Data Analysis (EDA)
- Statistical modeling and hypothesis testing
- Supervised learning (classification and regression)
- Ensemble methods
- Feature engineering and selection
- Model evaluation and validation

All analysis was performed using **Python, pandas, scikit-learn, matplotlib, and seaborn** in Jupyter notebooks.

---

## 📊 Projects

### 1. [EDA - Cardio Good Fitness Case Study](EDA%20-%20Cardio%20Good%20Fitness%20Case%20Study)
**Exploratory Data Analysis | Customer Profiling**

**Objective:** Analyze customer characteristics to inform product recommendations for different treadmill models.

**Key Techniques:**
- Univariate and bivariate analysis
- Distribution analysis (continuous and categorical variables)
- Customer segmentation
- Descriptive statistics and visualization

**Business Impact:** Identified distinct customer profiles for targeted marketing strategies.

**Dataset:** CardioGoodFitness.csv (customer demographics and treadmill purchase data)

---

### 2. [Linear Regression - Cars4U Case Study](Linear%20Regression%20-%20Cars4U%20Case%20Study)
**Regression Analysis | Pricing Model**

**Objective:** Build a pricing model for used cars to help Cars4U optimize their inventory pricing.

**Key Techniques:**
- Linear regression modeling
- Feature engineering (mileage, age, brand)
- Multicollinearity detection (VIF)
- Residual analysis and model diagnostics
- Train-test split and cross-validation

**Business Impact:** Developed data-driven pricing strategy for used car inventory.

**Dataset:** used_cars_data.csv (car specifications and pricing)

---

### 3. [Supervised Learning (Classification) - Personal Loan Campaign](Supervised%20Learning%20(Classification)%20-%20Personal%20Loan%20Case%20Study)
**Classification | Customer Targeting**

**Objective:** Predict which customers are likely to accept personal loan offers to improve campaign efficiency.

**Key Techniques:**
- Logistic regression
- Decision trees
- Model comparison and selection
- Confusion matrix analysis
- Precision, recall, and F1-score optimization
- Class imbalance handling

**Business Impact:** Improved loan campaign targeting, reducing marketing costs while increasing conversion.

**Dataset:** Loan_Modelling.csv (customer banking behavior and loan acceptance)

---

### 4. [Supervised Learning (Regression) - Anime Ratings](Supervised%20Learning%20(Regression)%20-%20Anime%20Ratings)
**Regression Analysis | Rating Prediction**

**Objective:** Build a model to predict anime ratings based on viewer behavior and anime characteristics.

**Key Techniques:**
- Multiple regression techniques
- Feature selection and engineering
- Regularization (Ridge, Lasso)
- Model performance comparison (R², RMSE, MAE)
- Hyperparameter tuning

**Dataset:** anime_data_raw.csv (anime metadata and user ratings)

---

### 5. [Ensemble Techniques - Travel Package Prediction](Ensemble%20Techniques%20-%20Travel%20Package%20Prediction)
**Ensemble Methods | Classification**

**Objective:** Predict which customers will purchase travel packages to optimize marketing spend.

**Key Techniques:**
- Random Forest
- Gradient Boosting (XGBoost, AdaBoost)
- Bagging and boosting comparison
- Feature importance analysis
- Ensemble model tuning
- ROC-AUC optimization

**Business Impact:** Identified high-value customer segments for targeted travel package marketing.

**Dataset:** Tourism.xlsx (customer demographics and travel package purchase history)
