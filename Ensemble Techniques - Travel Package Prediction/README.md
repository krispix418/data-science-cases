## Context
A mock tourism company offers 5 travel packages (Basic, Standard, Deluxe, Super Deluxe, King) and is launching a new Wellness Tourism Package. They want to target the right existing and potential customers to maximize uptake of the new product.

## Objective
- Use ensemble methods to predict which customers are likely to purchase the new travel package
- Identify the most important customer attributes driving purchase likelihood
- Compare ensemble techniques (bagging, boosting, stacking) to find the best performing model

## Techniques Used
- Decision Trees (baseline)
- Bagging / Random Forest
- AdaBoost
- Gradient Boosting
- XGBoost
- Stacking Classifier
- Feature importance analysis
- ROC-AUC optimization and model comparison

## Dataset
`Tourism.xlsx` — customer demographics, contact history, trip behavior, and package purchase outcomes (20 features, ~4,800 customers)
